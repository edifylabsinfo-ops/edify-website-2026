# TECH STACK & TỰ ĐỘNG HÓA

- **Core:** React (Vite) + Tailwind CSS.
- **Animations:** Thuần CSS với class `.reveal` (kết hợp Intersection Observer hook) và Javascript Canvas (`useCanvasEffect.js`).
- **Data Flow:** Quản lý state cục bộ bằng `useState`, `useEffect`. Không dùng Redux cho Landing Page MVP.
- **Responsive:** Mặc định Mobile-first. Bắt buộc có class `md:` hoặc `lg:` để tối ưu hiển thị trên màn hình lớn.
