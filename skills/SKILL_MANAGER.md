# EDIFY SKILL MANAGER — Evolution Logic

## 1. Định nghĩa Skill
Một "Skill" là một module code hoặc quy trình đã được tối ưu và QA thành công. 
Ví dụ: `skill-pricing-card`, `skill-hero-animation`.

## 2. Quy trình "Publish Skill"
Sau khi [Frontend Dev] hoàn thiện một component Pixel-perfect:
- Bước 1: Trích xuất logic và CSS chuẩn vào một file `.md` trong folder `skills/`.
- Bước 2: Ghi chú các tham số (Props) và cách sử dụng.
- Bước 3: Đăng ký kỹ năng này vào danh sách kỹ năng sẵn có.

## 3. Reusability First
Trước khi code mới, [Tech Lead] BẮT BUỘC phải check folder `skills/` xem đã có "bí kíp" tương tự chưa. Tuyệt đối không code lại từ đầu những thứ đã có Skill.
